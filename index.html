Python Stereo Vision Project

  • This project focused on stereo vision of an image. There were 3 different methods that we used; SSD algorithm which stands for Sum of Squared Difference. SAD which stands for Sum of Absolute Difference. NCC which stands for Normalized Cross Correlaton We would test on these 3 different methods using different templates and window sizes.

Below you will see many images. The first one is an image of the origanal picture from the left view and the second is the image from the right view. These two images will be used to create one image. The next 3 images will all be made using a template of size 10 and window of size 10. The first image is made using NCC, the second is with SAD, and the third is with SSD.

You can view the git repo here.
